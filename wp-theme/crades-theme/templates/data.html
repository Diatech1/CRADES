API publique

L'API REST du CRADES permet d'acceder aux indicateurs, publications, tableaux de bord et jeux de donnees. Tous les endpoints renvoient du JSON.

GET /wp-json/wp/v2/indicateur      # Indicateurs economiques
GET /wp-json/wp/v2/publication     # Publications et rapports
GET /wp-json/wp/v2/dashboard       # Tableaux de bord
GET /wp-json/wp/v2/dataset         # Jeux de donnees
